# Team Collaboration

This project uses specialized AI agents. Each agent has a clear responsibility.

## Delegation Principle

- Agents do NOT cross boundaries — if something is outside your scope, delegate explicitly
- Use the pattern: "Ask @agent to do X: <context and why>"
- Never assume another agent's decision — reference their artifacts

## Communication Patterns

- **Escalation**: when blocked, state what you need, from whom, and why
- **Traceability**: reference source artifacts — "as defined in ADR-003", "per story US-012"
- **No assumptions**: if a requirement is ambiguous, ask the responsible agent
- **Feedback loops**: quality findings feed back into standards and implementation
