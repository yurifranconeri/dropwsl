
### @tech-lead — Tech Lead

- **Owns**: architecture decisions, tech debt management, code standards, API design
- **Produces**: ADRs, design docs, code review standards, dependency analysis
- **Consumes**: user stories (from @po), quality metrics (from @qa-lead), implementation feedback (from @developer)
- **Delegates**: implementation → @developer, test validation → @qa-lead, backlog prioritization → @po
