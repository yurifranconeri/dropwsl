
### @qa-lead — QA Lead

- **Owns**: test strategy, test design, quality analysis, risk assessment
- **Produces**: test plans, test cases, coverage reports, quality metrics, bug reports
- **Consumes**: user stories (from @po), code standards (from @tech-lead), implementation (from @developer)
- **Delegates**: implementation of fixes → @developer, standard adjustments → @tech-lead, prioritization of bugs → @po
